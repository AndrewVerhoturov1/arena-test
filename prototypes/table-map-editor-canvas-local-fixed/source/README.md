# Редактор стола и карты 0

Локальная версия прототипа, собранная из предоставленного JSX-файла.

## Как запускать обычному пользователю

1. Распакуйте архив в папку. Не запускайте `.bat` прямо из окна zip-архива.
2. Дважды нажмите `Запустить.bat`. Если Windows плохо отображает русские имена файлов, используйте `start.bat`.
3. Откроется браузер с локальным адресом вида `http://127.0.0.1:5173/`.

`Запустить.bat` не делает `npm install`, не требует Node.js и не обращается к внутренним npm-кэшам. Он запускает уже готовую сборку из папки `dist` через маленький локальный PowerShell-сервер.

## Режим разработки

Если установлен Node.js, можно запустить `Запустить_через_Node.bat`. Этот режим нужен только для разработки. Зависимости будут устанавливаться из публичного npm registry.

## Важно

В архиве специально нет `package-lock.json`. Lock-файл, созданный в закрытой среде сборки, может содержать внутренние npm-cache ссылки и ломать установку на обычном компьютере.
