# Редактор стола и карты 0.2

Локальный браузерный прототип редактора стола/карты для 2D point-to-point настольных варгеймов.

## Что внутри

- Vite + React
- без backend
- без базы данных
- без авторизации
- весь интерфейс на русском языке

## Как запустить

Нужен установленный Node.js 18+.

```bash
npm install
npm run dev
```

После запуска открой адрес, который покажет Vite, обычно:

```text
http://localhost:5173/
```

## Сборка production-версии

```bash
npm run build
npm run preview
```

## Основные функции

- добавить свою карту-изображение;
- изображение загружается в родном разрешении;
- добавить несколько тестовых карт;
- выбрать карту кликом или через панель слоёв;
- перемещать карту мышью;
- менять размер через угловые маркеры;
- вращать через верхнюю ручку;
- менять прозрачность;
- включать/выключать тень объекта;
- настраивать тень: X, Y, размытие, растяжение, непрозрачность;
- менять порядок слоёв drag-and-drop;
- визуальный порядок на столе реально меняется;
- zoom колесом мыши в точку курсора;
- перемещение вида средней кнопкой мыши или пробел + drag;
- undo/redo на 10 шагов;
- ползунок считается одним действием в истории;
- экспорт проекта в JSON;
- сохранение проекта в localStorage.

## Горячие клавиши

- `Ctrl/⌘ + Z` — отменить действие;
- `Ctrl/⌘ + Y` — повторить действие;
- `Ctrl/⌘ + Shift + Z` — повторить действие;
- `Escape` — снять выделение;
- стрелки — сдвинуть выбранный объект на 0.5 px;
- `Ctrl/⌘ + стрелки` — сдвинуть на 0.1 px;
- `Shift + стрелки` — сдвинуть на 3 px;
- `Alt + ←/→` — повернуть на 0.25°;
- `Shift` при resize — сохранить пропорции;
- `Shift` при вращении — шаг 15°.

## Ограничения прототипа

- JSON импорт пока не реализован, есть экспорт и localStorage-save.
- Загруженные изображения хранятся как data URL в памяти браузера; для большого проекта позже лучше сделать asset-хранилище.
- Это редактор стола/карты, а не rules engine и не игровая реализация.
